# Sherlock 13 Online Game
A online game project in C language with SDL2 graphics for 4 players.

## Installation
- Compile:
```bash
./cmd.sh
```
- Launch server:
```bash
./server <Port>
```
- Launch clients:
```bash
./sh13 <IP address server> <Server port> <IP address client> <Client port> <Username>
```
